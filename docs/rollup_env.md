# Rollup Environment Variables

- `OVERLAY_URL` - endpoint for the overlay control plane.
- `VOICE_URL` - endpoint for the voice gateway.
- `OPA_URL` - OPA policy API endpoint.
- `NATS_URL` - JetStream endpoint.
- `VOICE_QOS_DSCP` - DSCP value for voice traffic (default 46).
- `VOICE_CODEC` - default codec (opus).
- `OVERLAY_DEFAULT_MTU` - default MTU for overlays (1600).
